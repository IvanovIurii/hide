# Hide утилита

## Описание

Утилита скрывающая файл, заданный в качестве аргумента, в "темном" каталоге.

### Как запускать

1. Сборка

Исполнить:
```
g++ -std=c++17 -o hide main.cpp
```

2. Запуск

Исполнить команду в нужно директории:
```
./hide <file_name>
```

### Дополнительно

Решил сделать без использования `filesystem` lib.
